## 	Proyectos destacados
# Automatización Educativa: Sistema de Registro de Incidencias Estudiantiles

- Un sistema automatizado de registro y notificación de incidencias estudiantiles, desarrollado sobre la plataforma de automatización n8n. Esta solución elimina      las tareas repetitivas del proceso, garantiza que el responsable legal sea notificado en tiempo real y mantiene un registro centralizado y auditable de todos       los eventos. El sistema es accesible desde cualquier dispositivo con conexión a internet, no requiere conocimientos de programación para su operación diaria y      puede adaptarse a las necesidades específicas de cualquier institución educativa de nivel básico o medio.
